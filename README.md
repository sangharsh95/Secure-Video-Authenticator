# Secure-Video-Authenticator
Tech Stack: Python, Django, SQLite3, HTML5, CSS3
MVT Framework Implementation: Developed a secure media platform using the Django framework, featuring a centralized videoauth configuration and a specialized videos application for media logic.
Secure Authentication Flow: Engineered custom backend views and templates, including login.html and base.html, to manage user sessions and enforce media access control.
Media Management System: Architected asynchronous upload and retrieval workflows with video_upload.html and video_list.html, ensuring data persistence via models.py and db.sqlite3.
